# Käyttöohje

Lataa tiedosto [CalculatorApp.jar](https://github.com/014587289/otm-harjoitustyo/releases/tag/0.3)

## Konfigurointi

Ohjelma ei tarvitse sen suurempia konfigurointeja. Sovelluksen pitäisi pystyä luomaan tallennukseen käytettävä tekstitiedosto, jos sellaista ei jo valmiiksi ole.

## Ohjelman käynnistys

Ohjelma käynnistetään komennolla 

'''
java -jar CalculatorApp.jar
'''

## Sovelluksen käyttö

Sovelluksen käyttö tapahtuu kertomalla inputtina sovellukselle, mitä käyttäjä haluaa tehdä. Alkuvalikosta löytyy vaihtoehdot laskutoimitusten tekemiselle,
 laskuhistorian tulostamiselle, sen tyhjentämiselle sekä sovelluksen sulkemiselle. Kun käyttäjä antaa inputtia, ohjeistaa sovellus käyttäjää, miten seuraavaksi menetellään. 
Esimerkiksi pluslaskua tehdessä tarvitaan käyttäjältä syötteenä kaksi lukua, jotka halutaan laskea yhteen.
